## PDF_Rename_Expert

Професійний інструмент для автоматизації перейменування відсканованих документів PDF за даними з Excel-бази.

**Інструкція для користувача** (встановлення Python, бібліотек, робота з програмою): [INSTRUKTSIYA_KORYSTUVACHA.md](INSTRUKTSIYA_KORYSTUVACHA.md).

### Нові можливості:
- **Вбудований перегляд**: Миттєве відображення всіх сторінок PDF прямо у вікні програми.
- **Підтримка АСКОЕ**: Динамічні поля для IP-адреси та номера модему.
- **Розумний експорт**: Формування звітів Excel з автоматичною нумерацією та налаштуванням ширини стовпців.
- **Лічильник символів**: Контроль довжини імені файлу для запобігання помилкам файлової системи.

### Структура проєкту (файли коду)
- `pdf_rename_expert.py`: Точка входу (запуск програми та збірка exe).
- `app_ui.py`: Вікно програми та логіка інтерфейсу.
- `naming.py`: Формування й розбір імен PDF, коди полів для експорту.
- `excel_db.py`: Читання Excel-бази споживачів.
- `Scan\Населення\`: Папка для вхідних PDF (Населення).
- `Scan\Промислові\`: Папка для вхідних PDF (Промислові).
- `pdf_rename_expert.log`: Журнал роботи програми (створюється автоматично).

### ⚠️ ПІДТРИМКА WINDOWS 7 (ВАЖЛИВО)

Щоб програма працювала на Windows 7, ви **ПОВИННІ** використовувати **Python 3.8.10**. Новіші версії Python (3.9+) на Windows 7 не запускаються.

#### Покрокова інструкція для збірки на Windows 7:

1. **Встановлення Python**:
   - Завантажте [Python 3.8.10 (Windows x86-64 executable installer)](https://www.python.org/downloads/release/python-3810/).
   - При встановленні обов'язково поставте галочку **"Add Python 3.8 to PATH"**.
   
2. **Встановлення залежностей**:
   Для звичайного запуску з коду достатньо рантайм-залежностей:
   ```bash
   pip install -r requirements.txt
   ```
   Щоб зібрати `.exe`, додатково встановіть інструменти розробки (зокрема PyInstaller):
   ```bash
   pip install -r requirements-dev.txt
   ```
   Або одразу обидва файли: `pip install -r requirements.txt -r requirements-dev.txt`.

3. **Компіляція в .exe**:
   Рекомендовано використовувати готовий spec-файл (підхоплює всі модулі проєкту):
   ```bash
   pyinstaller --noconfirm PDF_Rename_Expert.spec
   ```
   На Windows 7 зручніше виконати **`build_win7.bat`** (він ставить залежності з обох `requirements` і запускає PyInstaller).

4. **Якщо виникають помилки на Windows 7**:
   - Переконайтеся, що встановлені всі оновлення Windows (зокрема **KB2999226**).
   - Встановіть [Microsoft Visual C++ Redistributable](https://aka.ms/vs/16/release/vc_redist.x64.exe).
